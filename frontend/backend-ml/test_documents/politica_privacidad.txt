3. DATOS RECOPILADOS

Recopilamos los siguientes tipos de información:

- Datos de contacto: correo electrónico, teléfono, dirección

4. FINALIDAD DEL TRATAMIENTO

Los datos personales serán utilizados para:
- Gestión de la relación comercial
- Envío de comunicaciones promocionales (previo consentimiento)
- Mejora de nuestros servicios
- Cumplimiento de obligaciones legales

5. LEGITIMACIÓN

La base legal para el tratamiento de datos es:
- Ejecución de contrato
- Consentimiento expreso del usuario
- Interés legítimo
- Cumplimiento de obligaciones legales

6. CONSERVACIÓN DE DATOS

Los datos se conservarán durante:
- Relación comercial activa: mientras dure la relación
- Finalización de relación: 5 años para cumplimiento de obligaciones fiscales
- Consentimiento para comunicaciones: hasta su revocación

7. DERECHOS DE LOS USUARIOS

Los usuarios tienen derecho a:
- Acceso a sus datos personales
- Rectificación de datos inexactos
- Supresión de datos (derecho al olvido)
- Limitación del tratamiento
- Portabilidad de datos
- Oposición al tratamiento

Para ejercer estos derechos, contactar a: privacidad@empresaxyz.com

8. MEDIDAS DE SEGURIDAD

Implementamos las siguientes medidas de seguridad:
- Cifrado de datos sensibles
- Firewalls y sistemas de detección de intrusiones
- Auditorías de seguridad periódicas
- Formación continua del personal en protección de datos

9. TRANSFERENCIAS INTERNACIONALES

Los datos no se transferirán a países fuera del Espacio Económico Europeo,
salvo que sea necesario y se garanticen las medidas de seguridad adecuadas.
